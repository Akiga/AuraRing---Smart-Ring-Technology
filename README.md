# AuraRing Frontend

A modern responsive landing page for AuraRing built with React, Vite and Tailwind CSS.

## Features

- Responsive Landing Page
- Modern UI
- Newsletter Subscription Form
- Email Validation
- Toast Notification
- Connected to Express REST API

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- Axios
- React Router DOM
- Sonner

---

## Installation

```bash
git clone https://github.com/Akiga/AuraRing---Smart-Ring-Technology.git

cd frontend

npm install

npm run dev
```
